# Student Course Management System

A menu-driven C++ console application that manages student records 
and course enrollments.

## Features
- Add / Remove / Search students
- Enroll students in courses (no duplicates)
- Sort students by GPA
- Search by name
- Save & Load data from file

## Tech Stack
- Language: C++
- Concepts: OOP, Inheritance, Encapsulation
- Containers: vector, set
- Algorithms: sort(), find_if()

## How to Run
g++ main.cpp -o program
./program
